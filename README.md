# bynaki.auth

My authentication.


## Publish

```bash
yarn build.tsc      # 일단 빌드한다
npm login           # 로그인
npm publish         # 프로젝트의 루트 위치에서, npm에 배포 등록한다
```

## Usage

```bash
yarn token user permission1 permission2 ...
```
